# useLocation()

A stable re-export of `useLocation` from `react-router-dom`. Returns the current location object with `pathname`, `search`, and `hash`.

```tsx
import { useLocation } from 'boltdocs/hooks'
```

---

## Return Value

```tsx
const { pathname, search, hash, state, key } = useLocation()
```

---

## Why It Exists

This re-export exists to provide a **stable API surface**. If the underlying router library changes in a future version, your code continues to work without import changes.

---

## Usage Example

```tsx
import { useLocation } from 'boltdocs/hooks'

function CurrentPath() {
  const { pathname } = useLocation()
  const isHome = pathname === '/' || pathname === ''

  return isHome ? <HomePage /> : <DocPage />
}
```
