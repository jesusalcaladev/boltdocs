# useBreadcrumbs()

Generates a breadcrumb trail based on the currently active route's group and title.

```tsx
import { useBreadcrumbs } from 'boltdocs/hooks'
```

---

## Return Value

```tsx
const {
  crumbs,        // Array<{ label: string; href?: string }>
  activeRoute,   // ComponentRoute | undefined
} = useBreadcrumbs()
```

If the current route has a `groupTitle`, the breadcrumbs will include both the group and the page. Otherwise, just the page title.

---

## Usage Example

```tsx
import { useBreadcrumbs } from 'boltdocs/hooks'

function Crumbs() {
  const { crumbs } = useBreadcrumbs()

  return (
    <nav className="flex gap-2 text-sm text-zinc-400">
      {crumbs.map((crumb, i) => (
        <span key={i} className="flex items-center gap-2">
          {i > 0 && <span>/</span>}
          {crumb.href ? (
            <a href={crumb.href} className="hover:text-white">{crumb.label}</a>
          ) : (
            <span>{crumb.label}</span>
          )}
        </span>
      ))}
    </nav>
  )
}
```
