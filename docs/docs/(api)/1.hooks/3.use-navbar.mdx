# useNavbar

`useNavbar` is the logic hook for building custom top navigation bars. It automatically handles site identity, localized links, and social media integration from your `boltdocs.config.ts`.

---

## Usage

Use `useNavbar` when building a bespoke `Navbar` using the primitives.

```tsx
import { useNavbar } from 'boltdocs/hooks'

function CustomHeader() {
  const { title, logo, links, social } = useNavbar()
  
  return (
    <header>
      <img src={logo} alt={title} />
      <span>{title}</span>
      
      <nav>
        {links.map(link => (
          <a key={link.href} href={link.href}>{link.label}</a>
        ))}
      </nav>
    </header>
  )
}
```

---

## Return Value

| Property | Type | Description |
| :--- | :--- | :--- |
| `title` | `string` | The translated site title. |
| `logo` | `string \| null` | The URL for the current theme's logo (light/dark aware). |
| `logoProps` | `object` | An object containing `alt`, `width`, and `height` for the logo. |
| `links` | `NavbarLink[]` | Array of processed navigation links from the config. |
| `social` | `SocialLink[]` | Array of social media links (GitHub, X, Discord, etc.). |
| `github` | `string \| null` | Full GitHub repository URL if configured. |
| `theme` | `'dark' \| 'light'` | The current UI theme. |

---

## Automatic Theme Switching

The `logo` property in `useNavbar` is reactive. If your `boltdocs.config.ts` defines separate logos for light and dark modes, the hook will automatically return the correct URL when the theme changes.

```ts
// boltdocs.config.ts
export default {
  theme: {
    logo: {
      light: '/logo-light.svg',
      dark: '/logo-dark.svg',
      alt: 'My Brand',
    }
  }
}
```

---

## Localized Navigation

If `i18n` is enabled, `useNavbar` will automatically translate link labels using the current locale. Ensure your navbar configuration in `boltdocs.config.ts` follows the multi-language format:

```ts
// boltdocs.config.ts
export default {
  theme: {
    navbar: [
      {
        text: { en: 'Guides', es: 'GuĂ­as' },
        href: '/docs/guides'
      }
    ]
  }
}
```
