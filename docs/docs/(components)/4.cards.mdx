# Cards

Cards are versatile content blocks used to highlight features, project goals, or organized directory links. They are designed to work in a responsive grid layout using the `<Cards>` container.

## Variants & Examples

### Basic Usage

Wrap your individual `<Card>` components in a `<Cards>` container to create a clean grid.

<Cards cols={2}>
  <Card title="Getting Started" icon={<Rocket />} href="/docs/overview/installation">
    Quickly set up Boltdocs in your project.
  </Card>
  <Card title="Configuration" icon={<Settings />} href="/docs/overview/configuration">
    Learn how to customize every aspect of your site.
  </Card>
</Cards>

### Grid Layouts

You can adjust the `cols` prop to change how many cards appear per row on desktop screens.
