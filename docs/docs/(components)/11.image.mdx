# Image

The Image component provides a theme-aware way to render images that change based on whether the user is in Light or Dark mode. This is essential for screenshots, diagrams, and illustrations that need to match the current site aesthetics.

## Overview

<div style={{ display: 'grid', gridTemplateColumns: '1fr 1fr', gap: '1rem' }}>

## Variants & Examples

### Theme-Switching Images

By placing two `<Image>` components with different `theme` values back-to-back, you can create a single "logical" image that adapts to the user's theme.

<Image src="https://placehold.co/600x200/F4F4F4/1e1e1e?text=Themed+Creative" theme="light" alt="Light Asset" />
<Image src="https://placehold.co/600x200/1e1e1e/white?text=Themed+Creative" theme="dark" alt="Dark Asset" />

### Basic Usage

Standard Markdown `![]()` works perfectly in Boltdocs for global images. Use the `<Image>` component specifically when you need theme-awareness or advanced React logic.
