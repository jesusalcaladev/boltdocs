# Tabs

The Tabs component is used to organize content into multiple views, allowing users to toggle between related but distinct pieces of information within the same context. It features smooth sliding indicators, keyboard accessibility, and support for icons.

## Overview

<Tabs>
  <Tab label="Performance" icon={<Zap />}>
    Information about performance optimization and speed.
  </Tab>
  <Tab label="Security" icon={<Shield />}>
    Security best practices and system configuration.
  </Tab>
  <Tab label="CLI" icon={<Terminal />}>
    Reference for the Boltdocs command-line interface.
  </Tab>
</Tabs>

## Variants & Examples

### Basic Usage

Define a simple tabbed interface with just labels.

<Tabs>
  <Tab label="Installation">
    `pnpm add boltdocs`
  </Tab>
  <Tab label="Usage">
    `boltdocs dev`
  </Tab>
</Tabs>

### Package Manager Example

Tabs are commonly used to show installation instructions for different package managers.

<Tabs>
  <Tab label="npm">
    ```bash
    npm install boltdocs
    ```
  </Tab>
  <Tab label="pnpm">
    ```bash
    pnpm add boltdocs
    ```
  </Tab>
  <Tab label="bun">
    ```bash
    bun add boltdocs
    ```
  </Tab>
  <Tab label="deno" disabled>
    Deno support coming soon!
  </Tab>
</Tabs>
