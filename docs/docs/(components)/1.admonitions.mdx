# Admonitions (Callouts)

Admonitions highlight important information. Using them is as simple as adding an `<Admonition>` tag. Our callouts feature a modern, premium design with accent borders and glassmorphism.

**Using `<Admonition>`**

<Admonition type="info" title="Hey there!">
  This is a helpful note for the reader.
</Admonition>

<Admonition type="warning" title="Warning">
  Pay attention to this warning block.
</Admonition>

<Admonition type="important" title="Important Notice">
  This is an important piece of information that stands out.
</Admonition>

<Admonition type="caution" title="Caution">
  Be careful when performing this action.
</Admonition>

<Admonition type="danger" title="Danger">
  This is a destructive action notice.
</Admonition>

```tsx
<Admonition type="note">A basic note.</Admonition>
<Admonition type="tip">A helpful tip.</Admonition>
<Admonition type="info">General information.</Admonition>
<Admonition type="warning">A warning message.</Admonition>
<Admonition type="important">Extremely important info.</Admonition>
<Admonition type="caution">Exercise caution.</Admonition>
<Admonition type="danger">Danger / Critical error.</Admonition>
```

**Props (`AdmonitionProps`)**

<Field name="type" type='"note" | "tip" | "info" | "warning" | "important" | "caution" | "danger"' defaultValue='"note"'>
  The visual variant of the callout.
</Field>

<Field name="title" type="string">
  Optional. The title displayed at the top of the admonition.
</Field>

> **Note**: You can also use explicit tags like `<Note>`, `<Tip>`, `<Warning>`, `<Danger>`, `<InfoBox>`, `<Important>`, or `<Caution>` as shorthands.

## Personalization (CSS)

You can customize the look of each admonition type by overriding these CSS variables in your `custom.css`:

```css
:root {
  /* Info / Note */
  --ld-ui-info-bg: rgba(59, 130, 246, 0.08);
  --ld-ui-info-text: #60a5fa;
  --ld-ui-info-border: #3b82f6;

  /* Success / Tip */
  --ld-ui-success-bg: rgba(34, 197, 94, 0.08);
  --ld-ui-success-text: #4ade80;
  --ld-ui-success-border: #22c55e;

  /* Warning / Caution */
  --ld-ui-warning-bg: rgba(234, 179, 8, 0.08);
  --ld-ui-warning-text: #facc15;
  --ld-ui-warning-border: #eab308;

  /* Danger */
  --ld-ui-danger-bg: rgba(239, 68, 68, 0.08);
  --ld-ui-danger-text: #f87171;
  --ld-ui-danger-border: #ef4444;
}
```
