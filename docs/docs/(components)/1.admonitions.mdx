# Admonitions

Admonitions (also known as callouts) are high-visibility blocks used to highlight important information, tips, warnings, or errors. They feature a modern, glassmorphic design with accent borders and context-aware icons.

## Overview

<div style={{ display: 'flex', flexDirection: 'column', gap: '1rem' }}>
  <Admonition type="info" title="Pro Tip">
    Use admonitions sparingly to maintain their visual impact on the reader.
  </Admonition>

  <Warning title="System Update">
    Please ensure you have backed up your configuration before proceeding.
  </Warning>
</div>

## Variants & Examples

### Basic Usage

The simplest way to use an admonition is with the `<Admonition>` component. It defaults to the `note` type.

<Admonition>
  This is a default note admonition.
</Admonition>

### Shorthand Tags

Boltdocs provides several shorthand tags for the most common admonition types. These are equivalent to setting the `type` prop on the base `<Admonition>` component.

<Cards>
  <Note>Informational note.</Note>
  <Tip>Helpful tip or best practice.</Tip>
  <Warning>Potential issue to watch out for.</Warning>
  <Danger>Critical error or dangerous action.</Danger>
</Cards>

### Personalization (CSS)

You can globally customize the theme of your callouts by overriding the following CSS variables in your `custom.css`.

#### Info & Notes
```css
--ld-ui-info-bg: rgba(59, 130, 246, 0.08);
--ld-ui-info-text: #60a5fa;
--ld-ui-info-border: #3b82f6;
```

#### Success & Tips
```css
--ld-ui-success-bg: rgba(34, 197, 94, 0.08);
--ld-ui-success-text: #4ade80;
--ld-ui-success-border: #22c55e;
```

#### Warning & Caution
```css
--ld-ui-warning-bg: rgba(234, 179, 8, 0.08);
--ld-ui-warning-text: #facc15;
--ld-ui-warning-border: #eab308;
```

#### Danger
```css
--ld-ui-danger-bg: rgba(239, 68, 68, 0.08);
--ld-ui-danger-text: #f87171;
--ld-ui-danger-border: #ef4444;
```
