# Video

The Video component is a high-performance, lazy-loaded wrapper for the HTML5 `<video>` element. It uses an Intersection Observer to delay rendering until the component enters the viewport, significantly reducing initial page load weight.

## Overview

<Video
  src="https://www.w3schools.com/html/mov_bbb.mp4"
  poster="https://www.w3schools.com/html/pic_trulli.jpg"

/>

## Variants & Examples

### Basic Usage

Define a simple video player with just the source URL.

<Video src="https://www.w3schools.com/html/mov_bbb.mp4" />

### With Custom Poster

Always provide a `poster` image to ensure your documentation looks premium even before the video loads.

<Video
  src="https://www.w3schools.com/html/mov_bbb.mp4"
  poster="https://www.w3schools.com/html/pic_trulli.jpg"
  alt="Sample Video Demonstration"
/>

### Multiple Sources

For maximum browser compatibility, you can pass standard `<source>` elements as children.

<Video poster="https://www.w3schools.com/html/pic_trulli.jpg">
  <source src="video.webm" type="video/webm" />
  <source src="video.mp4" type="video/mp4" />
</Video>
