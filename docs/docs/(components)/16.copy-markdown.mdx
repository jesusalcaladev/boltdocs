# CopyMarkdown

The `CopyMarkdown` component renders a button (with an optional dropdown) at the top of each documentation page. It allows users to copy the raw MDX content of the current page to their clipboard.

---

## Default Behavior

When enabled (the default), a "Copy Markdown" button appears in the content header. Clicking it copies the raw `.mdx` source of the currently displayed page.

---

## Configuration

Control this feature globally in `boltdocs.config.ts`:

```ts
export default defineConfig({
  theme: {
    // Disable entirely
    copyMarkdown: false,

    // Or customize
    copyMarkdown: {
      text: 'Copy Source',
      icon: 'ClipboardCopy',
    },
  },
})
```

<Field name="copyMarkdown" type="boolean | { text?: string; icon?: string }" defaultValue="true">
  Set to `false` to hide the button. Pass an object to customize the text and icon.
</Field>

## Custom Implementation

Override the built-in CopyMarkdown via `mdx-components.tsx`:

```tsx
// docs/mdx-components.tsx
export default {
  CopyMarkdown: ({ mdxRaw, route }: any) => (
    <button
      onClick={() => {
        if (mdxRaw) navigator.clipboard.writeText(mdxRaw)
      }}
      className="text-xs text-zinc-400 hover:text-white"
    >
      📋 Copy ({route?.title})
    </button>
  ),
}
```

The component automatically receives `mdxRaw` and `route` as props when used in the layout.
