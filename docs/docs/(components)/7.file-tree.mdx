# File Tree

The FileTree component provides a visual representation of project directory structures. It uses standard Markdown lists to define the hierarchy and automatically assigns appropriate icons for folders and common file types.

## Overview

<FileTree>
- boltdocs.config.ts // Main configuration
- docs/
  - index.mdx
  - guides/
    - 1.introduction.mdx
  - components/
    - Button.tsx
- package.json
- public/
  - logo.svg
</FileTree>

## Variants & Examples

### Basic Usage

Simply wrap a Markdown list with the `<FileTree>` component. To indicate a folder, add a trailing slash `/` to the name.

<FileTree>
- src/
  - index.ts
  - utils.js
</FileTree>

### Inline Comments

You can add descriptive annotations to any file or folder by using `//` or `#` after the name.

<FileTree>
- src/ // Application source code
  - components/ # UI elements
  - hooks/ // Custom React hooks
- vite.config.ts # Build configuration
</FileTree>
