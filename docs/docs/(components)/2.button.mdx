# Button

The Button component is a primary interaction element used for navigation, form submission, and call-to-actions. It automatically handles internal routing via React Router or renders as a standard anchor tag for external links.

## Variants & Examples

### Basic Usage

Use the `<Button>` component directly in your MDX. It defaults to the `primary` variant and `md` size.

<Button>Primary Action</Button>

### Semantic Variants

Buttons come in several semantic variants to help establish intent and hierarchy.

<div style={{ display: 'flex', gap: '1rem', flexWrap: 'wrap' }}>
  <Button variant="primary">Primary</Button>
  <Button variant="secondary">Secondary</Button>
  <Button variant="success">Success</Button>
  <Button variant="danger">Danger</Button>
  <Button variant="warning">Warning</Button>
  <Button variant="info">Info</Button>
  <Button variant="subtle">Subtle</Button>
  <Button variant="ghost">Ghost</Button>
  <Button variant="outline">Outline</Button>
  <Button variant="link">Link</Button>
</div>

### Custom Colors

You can use the `className` prop to apply specific brand colors with neon glow effects.

<div className="flex flex-wrap gap-4 mt-6">
  <Button className="bg-purple-400">Purple</Button>
  <Button className="bg-cyan-500">Cyan</Button>
  <Button className="bg-pink-500">Pink</Button>
  <Button className="bg-orange-500">Orange</Button>
  <Button className="bg-blue-500">Blue</Button>
</div>

### Icons

Add icons for better visual recognition. You can place them on the left or right.

<div className="flex flex-wrap gap-4 mt-6">
  <Button icon={<ArrowLeft />}>Go Back</Button>
  <Button icon={<ArrowRight />} iconPosition="right">Next Step</Button>
  <Button variant="ghost" icon={<ExternalLink />} iconPosition="right">Open App</Button>
</div>

### Rounded Corners

Control the corner radius using the `rounded` prop.

<div className="flex flex-wrap gap-4 mt-6">
  <Button rounded="none">None</Button>
  <Button rounded="sm">Small</Button>
  <Button rounded="md">Medium</Button>
  <Button rounded="lg">Large</Button>
  <Button rounded="full">Full</Button>
</div>
