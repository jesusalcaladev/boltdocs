# Code Block

The Code Block component is the heart of Boltdocs' technical documentation. It provides beautiful syntax highlighting powered by Shiki, integrated copy-to-clipboard functionality, and flexible configuration options through Markdown meta strings.

## Markdown Usage

Boltdocs automatically transforms standard Markdown code fences into the `CodeBlock` component. You can configure it using the "meta string" area after the language ID.

### Basic Code Block

````markdown
```typescript
const message = "Hello, Boltdocs!";
console.log(message);
```
````

### With Line Numbers

Add `lineNumbers` to the meta string to enable the gutter.

````markdown
```typescript lineNumbers
function greet(name: string) {
  return `Hello, ${name}!`;
}
```
````

```typescript lineNumbers
function greet(name: string) {
  return `Hello, ${name}!`;
}
```

### With Title and Language

Use `title="..."` to display a filename or description in the header.

````markdown
```rust title="main.rs"
fn main() {
    println!("Hello, world!");
}
```
````

### Word Wrap

Enable `wordWrap` to prevent horizontal scrolling on long lines.

````markdown
```text wordWrap
This is a very long line of text that will automatically wrap to the next line instead of creating a horizontal scrollbar.
```
````

## Language Support

Boltdocs comes bundled with high-performance syntax highlighting for the following languages:

- **Web**: `html`, `css`, `js`, `ts`, `tsx`, `json`
- **Systems**: `rust`, `bash`, `yaml`, `toml`
- **Markdown**: `markdown`, `mdx`

> [!TIP]
> You can request additional languages by opening an issue or contributing to our highlighter core.

## Theming

You can customize the code block themes in your `boltdocs.config.ts` file. Boltdocs supports both single themes and separate themes for light and dark modes.

### Configuration

```typescript title="boltdocs.config.ts"
export default defineConfig({
  theme: {
    codeTheme: {
      light: 'github-light',
      dark: 'tokyo-night', // Custom dark theme
    }
  }
})
```

### Supported Themes

Below is the list of popular themes supported by Boltdocs. You can use these values in your configuration.

```ts title="types.ts"
export type ShikiTheme =
  | 'github-dark'
  | 'github-light'
  | 'nord'
  | 'dracula'
  | 'tokyo-night'
  | 'one-dark-pro'
  | 'one-light'
  | (string & {})
```

## Features

### Advanced Line Matching
The line numbering system is perfectly aligned with the code text, ensuring that even wrapped lines maintain a consistent vertical flow with their respective numbers.

### Intelligent Language Icons
Boltdocs automatically detects the language and displays the appropriate icon (Typescript, Rust, Python, etc.) in the header for a professional look.

### Truncation Support
For large files, you can use `shouldTruncate` to keep your documentation concise while still providing the full source code.
