# Link

The Link component is the primary method for navigation within Boltdocs. It intelligently handles both internal documentation routes (for instant SPA transitions) and external web links (with secure defaults).

## Variants & Examples

### Internal Navigation

Internal links use React Router under the hood to provide instantaneous transitions without a full page reload.

<Link to="/docs/overview/installation">Go to Installation</Link>

### External Links

Links to external domains automatically receive `target="_blank"` and `rel="noopener noreferrer"` for security and improved user experience.

<Link to="https://react.dev">Read React Documentation</Link>

### Technical Features

- **SEO Optimized**: Renders as standard `<a>` tags for optimal crawlability.
- **Protocol Detection**: Automatically identifies `http`, `https`, `mailto`, and `tel` links.
- **Smart Tooltips**: Integrated with the Boltdocs preview engine for rich hover states.

### Personalization (CSS)

Customize the typography and hover effects of all links using these CSS variables in your `custom.css`.

```css
:root {
  /* Colors */
  --ld-link-color: var(--ld-brand-primary);
  --ld-link-hover-color: var(--ld-brand-secondary);
  
  /* Decorations */
  --ld-link-decoration: none;
  --ld-link-hover-decoration: underline;
  
  /* Transition */
  --ld-link-transition: color 0.2s ease;
}
```
