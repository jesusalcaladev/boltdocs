# MDX Components

Boltdocs ships with a rich library of React components that you can use directly in your `.mdx` files. They are automatically available — just use them in JSX syntax.

```mdx
import { Note, Button, Cards, Card } from 'boltdocs/client'

<Note>This is a callout</Note>
<Button href="/docs">Get Started</Button>
```

## Import Paths

All components are available from `boltdocs/client`:

```tsx
import { Note, Tip, Warning, Button, Card, Cards, Tabs, Tab } from 'boltdocs/client'
```

Or granularly from `boltdocs/mdx`:

```tsx
import { Table, Field, FileTree } from 'boltdocs/mdx'
```

<Tip>
  Want to create your own globally available components? See the [MDX Components guide](/docs/customization/mdx-components).
