# List

The List component provides enhanced styling for bulleted lists, featuring custom icon markers like checkmarks and arrows. It is ideal for feature highlights, checklists, or step-by-step guides.

## Variants & Examples

### Basic Usage

The List component wraps standard `<li>` elements. It defaults to the `default` browser list style if no variant is specified.

<List>
  <li>Item One</li>
  <li>Item Two</li>
</List>

### Semantic Markers

Use `checked` for features or completed tasks, and `arrow` for directional or sequential lists.

<Cards>
  <List variant="checked">
    <li>Everything is included</li>
    <li>No configuration required</li>
  </List>
  <List variant="arrow">
    <li>Follow the path</li>
    <li>Reach the goal</li>
  </List>
</Cards>
