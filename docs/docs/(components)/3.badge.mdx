# Badge

The Badge component is an inline UI element used for status indicators, category labels, or metadata tags. It features a modern pill-shaped design with accent backgrounds and glassmorphism.

## Variants & Examples

### Basic Usage

Use the `<Badge>` component directly in your MDX. It works perfectly within text sentences or as standalone tags.

### Personalization (CSS)

Customize the appearance of all badges globally by overriding these CSS variables in your `custom.css`.

```css
:root {
  /* Success Badge */
  --ld-badge-success-bg: rgba(34, 197, 94, 0.15);
  --ld-badge-success-text: #4ade80;

  /* Warning Badge */
  --ld-badge-warning-bg: rgba(234, 179, 8, 0.15);
  --ld-badge-warning-text: #facc15;

  /* Danger Badge */
  --ld-badge-danger-bg: rgba(239, 68, 68, 0.15);
  --ld-badge-danger-text: #f87171;
  
  /* Global Radius */
  --ld-badge-radius: 9999px;
}
```
