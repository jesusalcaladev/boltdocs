# Utility Primitives

Boltdocs provides a set of core UI primitives that serve as the building blocks for all other components. These are built using [React Aria Components](https://react-aria.adobe.com/getting-started) to ensure perfect accessibility.

---

## Button
A versatile button component with multiple variants.

```tsx
import { PrimitiveButton } from 'boltdocs/client'
```

<Field name="variant" type='"primary" | "secondary" | "ghost" | "danger"'>
  The visual style of the button.
</Field>

<Field name="size" type='"sm" | "md" | "lg"'>
  Horizontal and vertical padding.
</Field>

<Field name="onPress" type="() => void">
  Callback for click/tap interaction.
</Field>

---

## Link
A smart link that automatically handles internal (client-side) vs external navigation.

```tsx
import { PrimitiveLink } from 'boltdocs/client'
```

<Field name="href" type="string" required>
  The target destination.
</Field>

<Field name="target" type='"_blank" | "_self"'>
  Standard HTML target attribute.
</Field>

---

## Tooltip & Popover
Floating UI elements for providing additional context.

```tsx
import { PrimitiveTooltip, PrimitivePopover } from 'boltdocs/client'
```

### Tooltip API
<Field name="content" type="ReactNode">
  The text or element to show on hover.
</Field>

<Field name="placement" type='"top" | "bottom" | "left" | "right"'>
  Where the tooltip should appear relative to the trigger.
</Field>

---

## Skeleton
A loading placeholder that mimics the layout of your final content.

```tsx
import { PrimitiveSkeleton } from 'boltdocs/client'
```

<Field name="className" type="string">
  Use `h-4 w-full` to control the dimensions.
</Field>

## Menu
A high-level accessible dropdown menu.

```tsx
import { PrimitiveMenu } from 'boltdocs/client'
```

### Full Example
```tsx
<PrimitiveMenu.Trigger>
  <button>Options</button>
  <PrimitiveMenu.Root>
    <PrimitiveMenu.Item onAction={() => console.log('Edit')}>Edit</PrimitiveMenu.Item>
    <PrimitiveMenu.Item onAction={() => console.log('Delete')}>Delete</PrimitiveMenu.Item>
  </PrimitiveMenu.Root>
</PrimitiveMenu.Trigger>
```
