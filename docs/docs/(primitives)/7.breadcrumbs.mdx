# Breadcrumbs

The `Breadcrumbs` primitive provides a path-based navigation trail. It's built on top of [React Aria](https://react-aria.adobe.com/Breadcrumbs) for full keyboard accessibility and screen reader support.

---

## Architecture

Breadcrumbs are usually placed at the very top of the content area to help users understand where they are.

```tsx
import { PrimitiveBreadcrumbs } from 'boltdocs/client'
```

---

## API Reference

### PrimitiveBreadcrumbs.Root
The main container for the breadcrumb trail.

<Field name="children" type="ReactNode">
  A list of `BreadcrumbsItem` components.
</Field>

### PrimitiveBreadcrumbs.Item
A single segment in the breadcrumb trail. It handles the list item (`<li>`) semantics.

### PrimitiveBreadcrumbs.Link
The clickable part of the breadcrumb. It automatically highlights the current page and disables navigation for it.

<Field name="href" type="string">
  The destination path. If omitted, it renders as a plain text label.
</Field>

### PrimitiveBreadcrumbs.Separator
The icon rendered between items (typically a `ChevronRight`).

## Building from Scratch

Using the `useBreadcrumbs` hook, you can automatically generate a trail based on the current page's grouping.

```tsx
import { PrimitiveBreadcrumbs } from 'boltdocs/client'
import { useBreadcrumbs } from 'boltdocs/client'

export function CustomBreadcrumbs() {
  const { crumbs } = useBreadcrumbs()

  if (crumbs.length === 0) return null

  return (
    <PrimitiveBreadcrumbs.Root>
      {crumbs.map((crumb, i) => (
        <PrimitiveBreadcrumbs.Item key={i}>
          {/* 1. Add separator before all items except the first one */}
          {i > 0 && <PrimitiveBreadcrumbs.Separator />}
          
          {/* 2. Render link or label */}
          <PrimitiveBreadcrumbs.Link href={crumb.href}>
            {crumb.label}
          </PrimitiveBreadcrumbs.Link>
        </PrimitiveBreadcrumbs.Item>
      ))}
    </PrimitiveBreadcrumbs.Root>
  )
}
```

---

## Logic Hook: `useBreadcrumbs`

The `useBreadcrumbs` hook is a simple utility that inspects the current route and its group configuration to produce an array of labels and links.

### Return Value

| Property | Type | Description |
| :--- | :--- | :--- |
| `crumbs` | `Breadcrumb[]` | Array of entities with `label` and optional `href`. |
| `activeRoute` | `Route` | The current route object from `useRoutes`. |

<Tip>
  If your page doesn't belong to a group, `useBreadcrumbs` will only return the current page title.
</Tip>
