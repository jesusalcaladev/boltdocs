# Installation

## Quick Start

The fastest way to create a new Boltdocs project is with the scaffolding CLI:

```bash
# Using pnpm (recommended)
pnpm create boltdocs@latest

# Using npm
npx create-boltdocs@latest

# Using yarn
yarn create boltdocs
```

The CLI will walk you through a few prompts (project name, template) and generate a fully configured project. Once it's done:

```bash
cd my-docs
pnpm install
pnpm dev
```

Open your browser at `http://localhost:5173` and you're live.

## Manual Setup

If you prefer to add Boltdocs to an existing project, you can install it manually.

### 1. Install Dependencies

```bash
pnpm add boltdocs react react-dom
pnpm add -D typescript @types/react @types/react-dom
```

### 2. Create `boltdocs.config.ts`

Create a configuration file at the root of your project:

```ts
import { defineConfig } from 'boltdocs'

export default defineConfig({
  siteUrl: 'https://my-docs.com',
  theme: {
    title: 'My Docs',
    description: 'Documentation for my project.',
  },
})
```

### 3. Create Your First Page

Create a `docs/` directory and add an `index.mdx` file:

```mdx
---
title: "Welcome"
description: "Welcome to the documentation."
---

# Welcome

This is your first documentation page.
```

### 4. Add Scripts

Add these scripts to your `package.json`:

```json
{
  "scripts": {
    "dev": "boltdocs dev",
    "build": "boltdocs build",
    "preview": "boltdocs preview"
  }
}
```

### 5. Start the Dev Server

```bash
pnpm dev
```

<Note>
  Boltdocs generates its own HTML entry point and Vite configuration — you don't need a `vite.config.ts` or `index.html` file when using the CLI commands.
</Note>
