# What is Boltdocs?

Boltdocs is the modern documentation engine built on the raw speed of **Vite** and the flexibility of **React**. It transforms your Markdown files into high-performance, interactive experiences without the usual complexity. 

## Why Boltdocs?

Documentation tools often force you to choose between simplicity and power. Boltdocs gives you both:

- **Built for Developers**: Use React components directly in your MDX. If you can build it in React, you can put it in your docs.
- **Instant Productivity**: Powered by Vite, development starts in milliseconds. Hot Module Replacement (HMR) keeps your flow state intact.
- **Batteries-Included**: Internationalization, versioning, dark mode, and search are ready to go from day one.

## Core Features

- ⚡️ **Blazing Speed**: Leveraging smart preloading and asset optimization for a snappy user experience.
- 📁 **Intuitive Routing**: Your folder structure *is* your site structure.
- 🎨 **Effortless Styling**: Customize the look and feel in seconds using [CSS variables](/docs/guides/customization).
- 🌍 **Global Reach**: Multilingual support is as simple as creating nested folders.
- 📚 **Version History**: Switch between different documentation versions seamlessly.
- 🧩 **Extensible**: Hook into the Vite lifecycle to add custom logic or transformations.
