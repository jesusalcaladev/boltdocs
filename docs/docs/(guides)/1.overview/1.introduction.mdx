# What is Boltdocs?

Boltdocs is the modern documentation engine you've been waiting for. Built on the raw speed of **Vite** and the flexibility of **React**, it transforms your Markdown files into high-performance, interactive experiences without the usual complexity. 

Whether you're building a simple API guide or a massive design system, Boltdocs stays out of your way while providing everything you need to shine.

## Why Boltdocs?

Documentation tools often force you to choose between simplicity and power. Boltdocs gives you both:

- **Built for Developers**: Use React components directly in your MDX. If you can build it in React, you can put it in your docs.
- **Instant Productivity**: Powered by Vite, development starts in milliseconds. Hot Module Replacement (HMR) keeps your flow state intact.
- **Batteries-Included**: Internationalization, versioning, dark mode, and search are ready to go from day one—no complex configuration required.

## Core Features

- ⚡️ **Blazing Speed**: Leveraging smart preloading and asset optimization for a snappy user experience.
- 📁 **Intuitive Routing**: Your folder structure *is* your site structure. It just works.
- 🎨 **Effortless Styling**: Customize the look and feel in seconds using CSS variables in `custom.css`.
- 🌍 **Global Reach**: Multilingual support is as simple as creating nested folders.
- 📚 **Version History**: Switch between different documentation versions seamlessly.
- 🧩 **Extensible**: Hook into the Vite lifecycle to add custom logic or transformations.
- 💅 **Rich Components**: Interactive `Playground`, `Admonitions`, `Tabs`, and beautiful **Link Previews** included.
