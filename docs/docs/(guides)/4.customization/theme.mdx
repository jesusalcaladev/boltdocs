# Theme Customization

Boltdocs gives you full control over the appearance of your documentation through **CSS Variables**. You never need to "eject" or modify framework internals — just override the variables in your `index.css` file.

---

## How It Works

Boltdocs defines all design tokens as CSS custom properties inside its default theme (`neutral.css`). The theme uses **Tailwind CSS v4's `@theme` directive** and semantic color aliases. To customize, import the base theme and override variables:

```css
/* index.css */
@import 'boltdocs/theme/neutral.css';

@theme {
  --color-primary-500: oklch(0.65 0.25 160);
  --font-sans: "Geist", system-ui, sans-serif;
}
```

<Important>
  Boltdocs uses **Tailwind CSS v4** syntax. Variables are defined inside `@theme { }` blocks, not `:root`. The `--color-*` prefix maps directly to Tailwind utility classes (e.g., `bg-bg-main` → `--color-bg-main`).
</Important>

### Dark Mode

Dark mode overrides are applied via the `:root[data-theme="dark"]` selector:

```css
:root[data-theme="dark"],
:root:not(.theme-light) {
  --color-bg-main: var(--color-neutral-950);
  --color-text-main: var(--color-neutral-50);
}
```

## Variable Reference

### Primary Color Scale

The primary color is defined as a 10-step OKLCH scale. Override these to change the accent color:

| Variable | Default | Usage |
| :--- | :--- | :--- |
| `--color-primary-50` | `oklch(0.985 0.012 265.8)` | Lightest tint |
| `--color-primary-100` | `oklch(0.92 0.045 265.8)` | Background tints |
| `--color-primary-200` | `oklch(0.84 0.085 265.8)` | Hover backgrounds |
| `--color-primary-300` | `oklch(0.74 0.125 265.8)` | Borders |
| `--color-primary-400` | `oklch(0.64 0.165 265.8)` | Links, inline code |
| `--color-primary-500` | `oklch(0.54 0.195 265.8)` | **Main accent** — buttons, active states |
| `--color-primary-600` | `oklch(0.45 0.165 265.8)` | Hover states |
| `--color-primary-700` | `oklch(0.36 0.135 265.8)` | Dark accents |
| `--color-primary-800` | `oklch(0.27 0.095 265.8)` | Very dark |
| `--color-primary-900` | `oklch(0.15 0.045 265.8)` | Darkest |

#### Changing the Primary Color

To set a green accent:

```css
@theme {
  --color-primary-50: oklch(0.98 0.02 155);
  --color-primary-100: oklch(0.93 0.05 155);
  --color-primary-200: oklch(0.85 0.1 155);
  --color-primary-300: oklch(0.75 0.15 155);
  --color-primary-400: oklch(0.65 0.2 155);
  --color-primary-500: oklch(0.55 0.22 155);
  --color-primary-600: oklch(0.45 0.18 155);
  --color-primary-700: oklch(0.35 0.14 155);
  --color-primary-800: oklch(0.25 0.1 155);
  --color-primary-900: oklch(0.15 0.05 155);
}
```

### Neutral Gray Scale

These control the base grays used for backgrounds, text, and borders:

| Variable | Used By |
| :--- | :--- |
| `--color-neutral-50` | Light mode `bg-main`, dark mode `text-main` |
| `--color-neutral-100` – `300` | Surfaces, borders, muted backgrounds |
| `--color-neutral-400` – `500` | Muted/dim text |
| `--color-neutral-700` – `800` | Dark mode borders, surfaces |
| `--color-neutral-900` – `950` | Dark mode backgrounds |

### Semantic Colors

These are mapped from the neutral/primary scales and control the UI:

| Variable | Description | Light Value | Dark Value |
| :--- | :--- | :--- | :--- |
| `--color-bg-main` | Base page background | `neutral-50` | `neutral-950` |
| `--color-bg-surface` | Cards, floating surfaces | `neutral-100` | `neutral-900` |
| `--color-bg-muted` | Hover/active backgrounds | `neutral-200` | `neutral-800` |
| `--color-text-main` | Primary text | `neutral-900` | `neutral-50` |
| `--color-text-muted` | Secondary text | `neutral-500` | `neutral-400` |
| `--color-text-dim` | Placeholders, metadata | `neutral-400` | `neutral-500` |
| `--color-border-subtle` | Light dividers | `neutral-200` | `neutral-800` |
| `--color-border-strong` | Strong dividers | `neutral-300` | `neutral-700` |
| `--color-code-bg` | Code block background | `neutral-100` | `neutral-900` |
| `--color-code-text` | Code block text | `neutral-800` | `neutral-200` |

### Layout Dimensions

| Variable | Default | Description |
| :--- | :--- | :--- |

| `--spacing-sidebar` | `16rem` | Width of the left sidebar |

| `--spacing-content-max` | `48rem` | Maximum width of the content area |

### Typography

| Variable | Default | Description |
| :--- | :--- | :--- |
| `--font-sans` | `Inter, system-ui, sans-serif` | Primary font stack |
| `--font-mono` | `JetBrains Mono, monospace` | Monospace font for code |
| `--font-display` | Same as `--font-sans` | Font for headings/display text |

#### Replacing the Font

```css
@import url('https://fonts.googleapis.com/css2?family=Outfit:wght@400;500;600;700&display=swap');

@theme {
  --font-sans: "Outfit", system-ui, -apple-system, sans-serif;
  --font-display: "Outfit", system-ui, -apple-system, sans-serif;
}
```

## Full Theme Override Example

```css
/* index.css */
@import 'boltdocs/theme/neutral.css';

/* Override primary to green */
@theme {
  --color-primary-500: oklch(0.65 0.2 155);
  --color-primary-400: oklch(0.72 0.18 155);
  --font-sans: "Geist", system-ui, sans-serif;
}

/* Customize dark mode */
:root[data-theme="dark"],
:root:not(.theme-light) {
  --color-bg-main: oklch(0.08 0.01 265);
  --color-bg-surface: oklch(0.12 0.01 265);
}
```
